# 📊 Predictive HR Analytics: Employee Attrition Model

An end-to-end exploratory data analysis and statistical profile modeling pipeline designed to evaluate workforce allocation trends, organizational dynamics, and structural attrition risks. This project uses an interactive Jupyter Notebook (`hr_attrition_analysis.ipynb`) to evaluate risk indicators across 14,999 employee records.

## 🛠️ Data Stack Architecture
* **NumPy:** Managed underlying vectorized calculations and math operations.
* **Pandas:** Applied for loading matrix structures, auditing operational indices, and isolating departmental counts.
* **Matplotlib & Seaborn:** Utilized to map statistical data fields, plotting generational proportions (Pie Charts) and conditional project frequencies (Countplots).

## 🗺️ Analytical Pipeline Phases
1. **Environment Setup:** Constructing safe compilation configurations and layout defaults.
2. **Data Acquisition:** Parsing operational employee metrics (`satisfactoryLevel`, `avgMonthlyHours`, `timeSpent.company`).
3. **Workforce Profiling:** Charting volume structures cleanly across corporate organizational business segments.
4. **Retention Auditing:** Cross-referencing operational project stress indices with historical resignation fields (`left`).
